ダウンロードいただきありがとうございます。
このゲームは学校の課題で制作したゲームなので色々な所がガバガバです。

白いロケットが飛んでいるだけなので何もやましい事は一切ありません。

//-------------------------------
//更新履歴
//-------------------------------
まだ一度もしてないよ

//-------------------------------
//操作方法
//-------------------------------
Enterキーでスタート
↓
Spaceキー連打でエネルギーチャージ
↓
Enterキーで角度をキメて発射！
以上！

リザルト画面からはEnterキーでタイトル画面に戻る事が出来ます。

//-------------------------------
// BGM
//-------------------------------
ロケットが飛んでいる最中のBGMは「i don't want to miss a thing」を使っていましたが著作的な面を考慮し
削除しました。
BGMを追加する場合は「RED ROCKET.exe」があるフォルダにwavファイルをぶち込んで「rocket.wav」に名前を
変更すれば反映されます。

やり方が分からない人や反映されない人は「アルマゲドン」のエンディングを脳内再生してください。
